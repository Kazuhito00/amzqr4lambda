# amzqr4lambda
[Amazing-QR](https://github.com/x-hw/amazing-qr)をAWS Lambda上で動かしたサンプルです。<br>
Amazing-QRを用いて画像ファイルやアニメGifを用いたQRコードが作成できます(埋め込める情報は半角英数字)<br><br>
